<p align="center">
<h1> Sistema de Banco de dados - Loja perfumaria</h1>

<h3>Projeto Acadêmico - Faculdade Cruzeiro do Sul</h3>
</p>
O projeto foi desenvolvido para implementar um banco de dados funcional para uma <strong>Loja de Perfumaria</strong>, permitindo registrar e manipular informações essenciais.
O sistema serve para demostrar como um banco opera com dados reais usando SQL.
</p>

</div>
<h3>Estruturas do projeto</h3>
<ul>
  <li><strong>Scripts SQL de criação das tabelas</strong> (estrutura do banco)</li>
  <li><strong>Scripts SQL de inserção de dados</strong> (povoamento das tabelas)</li>
  <li><strong>Consultas SELECT</strong> utilizando JOIN, WHERE E ORDER BY</li>
  <li><strong>Comandos UPDATE</strong> para atualização de registros</li>
  <li><strong>Comandos DELETE</strong> com condições de segurança</li>
</ul>
<div>

  <div>
    <h3> Organização e formas do projeto e repositório</h3>
    <p>
      O repositório foi estruturado utilizando o <strong>Git Bash e GitHub</strong> para versionamento, garantindo histórico, organização e segurança dos arquivos.
      Cada funcionalidade do banco foi separada em arquivos .sql individuais.
    </p>
  <h3> Comandos utilizados:</h3>
  <ul>
  <li><strong>CREATE TABLE</strong> - criação das entidades</li>
  <li><strong>INSERT INTO</strong> - inserção de registros</li>
  <li><strong>SELECT</strong> - consultas</li>
  <li><strong>JOIN</strong> - combinações entre tabelas</li>
  <li><strong>WHERE</strong> - filtro de consultas</li>
  <li><strong>ORDER BY</strong> - ordenação</li>
  <li><strong>UPDATE</strong> - alterações de dados</li>
  <li><strong>DELETE</strong> - exclusões condicionais</li>
  </ul>

</div>
<h3> Práticas desenvolvidas:</h3>
<ul>
<li>Criação de banco de dados estruturados</li>
<li>Modelagem e organização lógica em SQL</li>
<li>Povoamento e manipulação de tabelas</li>
<li>Consultas relacionais utilizando JOIN</li>
<li>Uso de UPDATE e DELETE com segurança</li>
 <li>Versionamento com Git Bash e organização no GitHub</li>
</ul>
